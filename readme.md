# Panchang Web App

A web-based Panchang calendar with real-time calculations and clean UI.

## 🔥 Features
- Daily Panchang (Tithi, Nakshatra, Yoga, Karana)
- Choghadia calculation
- Rashi & Horoscope section
- Multi-language support
- Share Panchang feature with wallpapers

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- PHP (custom Panchang calculation engine)
