How to use

  • Put your Prokerala client_id and client_secret inside rashifal-api.php.
  • Upload both files to the same folder on your server.
  • Select a rashi and prediction category, then click Get Rashifal.
  • This page uses the /horoscope/daily/advanced endpoint (v2 API). Weekly / Monthly / Yearly prediction endpoints are not available in API v2.
